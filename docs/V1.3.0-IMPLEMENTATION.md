**`agentSuggest.ts` resolution order**:

```ts
export async function suggestStack(goal: string): Promise<StackSuggestion> {
  // 1. Try active CLI subscription
  const statuses = await invoke<RuntimeStatus[]>("query_all_agent_statuses");
  const ready = statuses.find(s => s.connected && s.authenticated);
  if (ready) {
    return suggestViaCLI(ready.runtime, goal);  // shells out via promptAgent()
  }

  // 2. Try active API key
  const keys = await invoke<LlmApiKey[]>("list_llm_api_keys");
  const activeKey = keys.find(k => k.enabled);
  if (activeKey) {
    return suggestViaApiKey(activeKey, goal);  // direct provider SDK
  }

  // 3. Try Ollama
  const ollama = await invoke<OllamaStatus>("get_ollama_status");
  if (ollama.running && ollama.models.length > 0) {
    return suggestViaOllama(ollama, goal);
  }

  // 4. Hosted Pro fallback
  const auth = useAuthStore.getState();
  if (auth.isAuthenticated && auth.tier === "pro") {
    return suggestViaCloud(goal);  // POST /agent-suggest on ato-cloud
  }

  // 5. None available — throw, wizard shows "connect a runtime" link
  throw new Error("NO_RUNTIME_AVAILABLE");
}
```

**`agentWriter.ts` file-writing contract**:

```ts
export async function writeAgent(agent: Agent): Promise<void> {
  switch (agent.runtime) {
    case "claude":
      // ~/.claude/agents/<slug>.md (or <project>/.claude/agents/<slug>.md)
      await invoke("write_file_safe", {
        path: claudeAgentPath(agent),
        contents: renderClaudeAgent(agent),
        hashBefore: null,  // new file
      });
      break;
    case "codex":
      // ~/.codex/agents/<slug>/ + AGENTS.md
      await invoke("write_codex_agent", { agent });
      break;
    case "gemini":
      // <project>/.gemini/agents/<slug>.yaml + root_agent.yaml entry
      await invoke("write_gemini_agent", { agent });
      break;
    case "openclaw":
      // ~/.openclaw/agents/<slug>/ with SOUL.md + TOOLS.md
      await invoke("write_openclaw_agent", { agent });
      break;
    case "hermes":
      // ~/.hermes/agents/<slug>/
      await invoke("write_hermes_agent", { agent });
      break;
  }
  await invoke("create_agent_record", { agent });
}
```

All writes go through the existing safety pipeline: `write_file_safe` (already exists) does hash check + auto-backup + audit log. New per-runtime commands wrap it.

**Guided path state machine** (use Zustand or `useReducer`):

```
idle → asking_goal → suggesting → confirming → writing → done
                          ↑                ↓
                          └── editing ─────┘
```

Persist draft to `~/.ato/agent-drafts/<draftId>.json` after each transition so users can leave/resume.

**Quick path**: single-page form using `react-hook-form` + Zod schema. Fields per the PRD F1.B section. Submit calls the same `writeAgent()`.

**Acceptance**:
- Guided: free-text → suggestion cards → confirm → agent appears in Agents list with correct files on disk + SQLite record.
- Quick: same outcome, in <30s.
- Drafts persist; closing/reopening wizard resumes.
- All writes audit-logged + auto-backed-up.

**Tauri command surface**:

```rust
#[tauri::command]
async fn pty_spawn(cwd: String, shell: Option<String>) -> Result<String, String>;  // returns ptyId

#[tauri::command]
async fn pty_write(pty_id: String, data: String) -> Result<(), String>;

#[tauri::command]
async fn pty_resize(pty_id: String, rows: u16, cols: u16) -> Result<(), String>;

#[tauri::command]
async fn pty_kill(pty_id: String) -> Result<(), String>;
```

PTY data streams via Tauri events (`pty://data/<ptyId>` → `{data: string}`).

**Frontend**:

```tsx
const term = new Terminal({ fontSize: 13, theme: { background: "#0a0a0f" } });
const fitAddon = new FitAddon();
term.loadAddon(fitAddon);
term.open(containerRef.current);
fitAddon.fit();

const ptyId = await invoke<string>("pty_spawn", {
  cwd: activeProject?.path ?? "~",
  shell: null,  // defaults to $SHELL or pwsh.exe
});

await listen<{data: string}>(`pty://data/${ptyId}`, (e) => term.write(e.payload.data));
term.onData((data) => invoke("pty_write", { ptyId, data }));
term.onResize(({rows, cols}) => invoke("pty_resize", { ptyId, rows, cols }));
```

**Pane behavior**:
- Default collapsed (32px header strip).
- Click header to expand to 320px (resizable).
- Toggle Chat / Shell with two pill buttons in the header.
- Persist height + last tab to `localStorage`.
- Survives navigation (component lives in `Dashboard.tsx`, not per-section).

**Cross-feature**: when `agentSuggest` (T3) or MCP install (T4) wants to "show its work," it writes a status line + command to the active shell PTY: `[ATO] $ npx -y @modelcontextprotocol/server-filesystem /`.
